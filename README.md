# gym-ui-react

A modern and responsive gym website built using React.js.  
The project provides an attractive user interface to showcase gym services, trainers, programs, and contact form.

## 🚀 Features

- Responsive design for all devices
- Modern gym landing page
- Home, About, Services, Trainers, and Contact sections
- Contact form UI
- Smooth navigation
- Interactive user interface
- Clean and reusable React components

## 🛠️ Tech Stack

- React.js
- JavaScript
- HTML5
- CSS3
